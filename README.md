### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Quick Start
```bash
# Clone repository
git clone https://github.com/vihaankulkarni29/website-gen.git
cd website-gen

# Create new client project
./scripts/create-client.sh your-client-name

# Navigate to project

npm install

npm run dev
```

```bash

```
